VisuaLearn AI is an adaptive educational platform developed to improve how learners understand abstract and technical concepts through conversational guidance, visual learning artifacts, simulations, quizzes, flashcards, and persistent learning history. Many online learning systems still depend on static notes or generic chatbot responses, which can limit engagement, reduce personalization, and make visual or process-oriented topics difficult to understand. These limitations are addressed in this report through a full-stack learning platform that combines learner context, AI-assisted explanation, document-based retrieval, adaptive content generation, and structured revision support within a single web application.

The objective of this work is to design and implement a modular intelligent learning system that supports authenticated user sessions, conversational tutoring, personalized learning content, document-aware responses, feedback collection, and reusable educational artifacts. The system uses agent-based decision logic to identify learner intent, route requests, and generate appropriate learning outputs such as notes, mind maps, quizzes, flashcards, simulations, and video-oriented learning support. The frontend is implemented using React, Vite, Tailwind CSS, and reusable components, while the backend is implemented using Node.js and Express. Supabase provides authentication, PostgreSQL-based persistence, and storage services, while Azure OpenAI services support language generation, embeddings, and selected multimodal features.

The implemented platform was evaluated through functional testing and a preliminary user study involving 30 participants who interacted with learning modules such as explanations, visual artifacts, simulations, quizzes, and document-based learning. The testing procedure verified authentication, chat persistence, streaming responses, learning resource generation, retrieval-augmented document usage, feedback handling, and safe rendering of generated content. Participant feedback indicated that personalized responses, visual explanations, and simulations improved engagement and helped learners understand concepts more clearly than static learning material. The results show that VisuaLearn AI provides a practical foundation for an adaptive, interactive, and visually supported learning environment.
