\chapter{Testing and Results}

This chapter reports the testing and evaluation carried out for VisuaLearn AI. The work was examined through two complementary activities. The first activity checked whether the main software modules functioned correctly when used as part of a complete learning workflow. The second activity collected feedback from 30 participants after they used the platform for topic-based learning tasks. Together, these activities provided an initial view of system reliability, usability, learner perception, and the role of adaptive visual support in the application.

\section[Evaluation Methodology]{\textbf{Evaluation Methodology}}
The evaluation combined system validation with participant-based assessment. Functional testing covered authentication, conversation management, generation of learning resources, document-supported learning, simulation output, feedback submission, and data persistence. The participant assessment examined whether learners could use the platform to understand concepts, revise material, view visual explanations, and complete self-assessment activities.

The participant study followed a fixed sequence. Each participant selected one topic, completed a pre-learning activity, used VisuaLearn AI, and then completed a post-learning activity along with a feedback form. The selected topics were chosen because they represent different learning requirements. Some topics require conceptual clarity, some require procedural reasoning, and others benefit from visual explanation.

\section[Discussion of Results]{\textbf{Discussion of Results}}
The evaluation shows that VisuaLearn AI can function as an interactive learning support platform in a preliminary setting. Participants responded well to adaptive explanations, visual summaries, flashcards, quizzes, and simulations. The strongest feedback was related to usability and willingness to reuse the platform, which indicates that the interface and workflow were acceptable for learner interaction.

The findings also show that topic type affects the usefulness of each learning mode. Procedural topics such as CPU scheduling and deadlock detection were better suited to simulation and guided stepwise explanation. Conceptual topics such as blockchain, neural networks, and transformer attention benefited from concise explanations, examples, visual summaries, and revision artifacts.

There are limitations to the evaluation. The study involved 30 participants, which is appropriate for an initial project assessment but not enough for broad educational generalization. The interaction period was short, so the study did not measure long-term retention, classroom use, or performance across a wider learner population.

\section[Observed Outcomes]{\textbf{Observed Outcomes}}
The observed outcomes of the evaluation are summarized below:
\begin{itemize}[leftmargin=1.2cm]
	\item Participants used generated explanations, flashcards, and quizzes for revision and self-assessment.
	\item Participants gave positive feedback on mind maps and simulations as visual learning aids.
	\item Procedural topics were easier to inspect when guided steps or simulations were available.
	\item Conceptual topics were supported through concise explanations, examples, and organized revision material.
	\item Document-supported learning improved answer relevance when uploaded material was available.
	\item Persistent learning history allowed earlier conversations and generated resources to be reviewed later.
	\item Adaptive response selection contributed to a more personalized learning experience.
\end{itemize}

\section[Result Summary]{\textbf{Result Summary}}
The testing and evaluation show that VisuaLearn AI works as an integrated adaptive learning workspace. The system supports conversation, learning artifacts, document-based learning, simulations, feedback, and persistence. The preliminary study with 30 participants recorded positive responses for usability, personalization, visual learning support, and overall satisfaction. These results support the practical value of the implemented system, while further work is needed for larger participant groups, delayed retention testing, and more rigorous measurement of learning outcomes.
